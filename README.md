# Análise de Algoritmos

Este repositório contém as atividades desenvolvidas para a disciplina **Análise de Algoritmos**, do curso de **Ciência da Computação**. As implementações têm como objetivo aplicar conceitos teóricos estudados em aula, como análise de complexidade, recursão e comparação de desempenho entre algoritmos.

## Estrutura do Repositório

As atividades estão organizadas por conjuntos de deveres de casa. Cada diretório contém as atividades correspondentes ao respectivo dever.

```text
deveres-de-casa/
├── dever-1/
│   └── atividade_1/
├── dever-2/
│   └── atividade_2/
├── dever-3/
│   └── atividade_3/
```

Cada pasta de atividade pode conter:

* Implementações de algoritmos em **Python**
* **Notebooks Jupyter** utilizados para experimentos
* Análises de complexidade
* Resultados obtidos durante os testes

## Tecnologias Utilizadas

* Python 3
* Jupyter Notebook
* Bibliotecas padrão do Python (ex.: `time`, `math`, `random`)

## Conteúdos Trabalhados

As atividades abordam conceitos fundamentais da área de análise de algoritmos, incluindo:

* Análise assintótica
* Notação Big-O
* Algoritmos recursivos
* Comparação de desempenho entre algoritmos
* Experimentos computacionais

## Objetivo

O objetivo deste repositório é registrar as implementações e experimentos realizados durante a disciplina, servindo como material de apoio para estudo e acompanhamento do desenvolvimento das atividades.

## Observação

Este repositório possui finalidade **acadêmica e educacional**, sendo utilizado para armazenamento e organização das atividades da disciplina.
